# VideoDecoder.off(string eventName, function callback)
基础库 2.11.1 开始支持，低版本需做[兼容处理](../../../guide/runtime/client-lib/compatibility.html)。
## 功能描述
取消监听录制事件。当对应事件触发时，该回调函数不再执行
## 参数
### string eventName
事件名
### function callback
事件触发时执行的回调函数
