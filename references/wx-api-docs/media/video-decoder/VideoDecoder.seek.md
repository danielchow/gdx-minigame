# Promise VideoDecoder.seek(number position)
基础库 2.11.1 开始支持，低版本需做[兼容处理](../../../guide/runtime/client-lib/compatibility.html)。
## 功能描述
跳到某个时间点解码
## 参数
### number position
跳转的解码位置，单位 ms
## 返回值
### Promise
基础库 2.16.1 开始支持，低版本需做[兼容处理](../../../guide/runtime/client-lib/compatibility.html)。
