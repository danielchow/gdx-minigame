# Promise Video.requestFullScreen(number direction)
**微信 鸿蒙 OS 版**：支持
## 功能描述
视频全屏
## 参数
### number direction
设置全屏时视频的方向

**direction 的合法值**
 | 值 | 说明 | 最低版本 |
| --- | --- | --- |
| 0 | 正常竖向 |  |
| 90 | 屏幕逆时针90度 |  |
| -90 | 屏幕顺时针90度
## 返回值
### Promise
视频全屏完成的 Promise
