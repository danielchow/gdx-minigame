# WebAudioContextNode
一类音频处理模块，不同的Node具备不同的功能，如GainNode(音量调整)等。一个WebAudioContextNode可以通过上下文来创建。
目前已经支持以下Node：
IIRFilterNode
WaveShaperNode
ConstantSourceNode
ChannelMergerNode
OscillatorNode
GainNode
BiquadFilterNode
PeriodicWaveNode
BufferSourceNode
ChannelSplitterNode
ChannelMergerNode
DelayNode
DynamicsCompressorNode
ScriptProcessorNode
PannerNode
AnalyserNode
