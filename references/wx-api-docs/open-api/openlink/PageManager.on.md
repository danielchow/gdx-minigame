# PageManager.on(string eventName, function callback)
基础库 3.6.7 开始支持，低版本需做[兼容处理](../../../guide/runtime/client-lib/compatibility.html)。
## 功能描述
监听来自活动、功能向开发者产生的某些事件。
## 参数
### string eventName
事件名称，由渠道获得需要监听的事件名称
### function callback
发生某种事件时的回调函数指针
