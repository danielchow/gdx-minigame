# wx.offInteractiveStorageModified(function callback)
基础库 2.14.4 开始支持，低版本需做[兼容处理](../../../guide/runtime/client-lib/compatibility.html)。
## 功能描述
取消监听成功修改好友的互动型托管数据事件，该接口在游戏主域使用
## 参数
### function callback
事件发生的回调函数，留空则清除所有回调
