# Boolean wx.setHandoffQuery(String query)
基础库 2.14.4 开始支持，低版本需做[兼容处理](../../guide/runtime/client-lib/compatibility.html)。
## 功能描述
设置接力参数，该接口需要在游戏域调用
## 参数
### String query
需要传递给接力设备的参数，格式 为querystring
## 返回值
### Boolean
设置是否成功
