# boolean GameRecorder.isSoundSupported()
基础库 2.8.0 开始支持，低版本需做[兼容处理](../../guide/runtime/client-lib/compatibility.html)。
## 功能描述
获取是否在录制游戏画面的同时支持录制游戏音频的信息
## 返回值
### boolean
是否在录制游戏画面的同时支持录制游戏音频
