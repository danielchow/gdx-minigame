# GameRecorderShareButton.onTap(function listener)
基础库 2.8.0 开始支持，低版本需做[兼容处理](../../guide/runtime/client-lib/compatibility.html)。
## 功能描述
监听游戏对局回放分享按钮的点击事件。只有当分享由于非用户取消的原因失败时，该事件的回调函数才会执行。
## 参数
### function listener
游戏对局回放分享按钮的点击事件的监听函数
