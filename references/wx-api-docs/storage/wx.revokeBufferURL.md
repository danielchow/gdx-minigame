# wx.revokeBufferURL(string url)
基础库 2.14.0 开始支持，低版本需做[兼容处理](../../guide/runtime/client-lib/compatibility.html)。

**微信 Windows 版**：支持

**微信 Mac 版**：支持

**微信 鸿蒙 OS 版**：支持

相关文档: [存储策略](../../guide/base-ability/storage.html)
## 功能描述
根据 URL 销毁存在内存中的数据
## 参数
### string url
需要销毁的二进制数据 URL
