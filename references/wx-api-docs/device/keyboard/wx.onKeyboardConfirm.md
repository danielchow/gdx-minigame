# wx.onKeyboardConfirm(function listener)
**微信 鸿蒙 OS 版**：支持
## 功能描述
监听用户点击键盘 Confirm 按钮时的事件
## 参数
### function listener
用户点击键盘 Confirm 按钮时的事件的监听函数
#### 参数
##### Object res | 属性 | 类型 | 说明 |
| --- | --- | --- |
| value | string | 键盘输入的当前值 |
