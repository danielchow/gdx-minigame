# wx.offBluetoothAdapterStateChange()
基础库 2.9.2 开始支持，低版本需做[兼容处理](../../../guide/runtime/client-lib/compatibility.html)。

相关文档: [蓝牙介绍](../../../guide/device/bluetooth.html)
## 功能描述
移除蓝牙适配器状态变化事件的全部监听函数
## 示例代码

wx.offBluetoothAdapterStateChange()
