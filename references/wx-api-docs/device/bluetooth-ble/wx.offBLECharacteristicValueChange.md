# wx.offBLECharacteristicValueChange()
基础库 2.9.2 开始支持，低版本需做[兼容处理](../../../guide/runtime/client-lib/compatibility.html)。

相关文档: [蓝牙低功耗 (BLE)](../../../guide/device/ble.html)
## 功能描述
移除蓝牙低功耗设备的特征值变化事件的全部监听函数
## 示例代码
```js
wx.offBLECharacteristicValueChange()
```
