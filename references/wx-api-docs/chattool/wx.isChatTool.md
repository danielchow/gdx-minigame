# wx.isChatTool()
基础库 3.7.12 开始支持，低版本需做[兼容处理](../../guide/runtime/client-lib/compatibility.html)。

相关文档: [聊天工具模式](../../guide/open-ability/chat-tool.html)
## 功能描述
是否处于聊天工具模式
## 示例代码
```javascript
const isChatToll = wx.isChatTool()
  console.log(isChatToll) // 是否处于聊天工具模式
```
