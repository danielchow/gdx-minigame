# boolean wx.isPointerLocked()
基础库 3.2.0 开始支持，低版本需做[兼容处理](../../../guide/runtime/client-lib/compatibility.html)。

**微信 Windows 版**：支持

**微信 Mac 版**：支持
## 功能描述
检查鼠标指针是否被锁定。此接口仅在 Windows、Mac 端支持。
## 返回值
### boolean
是否被锁定
