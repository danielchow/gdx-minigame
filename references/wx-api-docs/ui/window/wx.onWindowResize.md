# wx.onWindowResize(function listener) ## # 功能描述
监听窗口尺寸变化事件
## 参数
### function listener
窗口尺寸变化事件的监听函数
#### 参数
##### Object res | 属性 | 类型 | 说明 |
| --- | --- | --- |
| windowWidth | number | 变化后的窗口宽度，单位 px |
| windowHeight | number | 变化后的窗口高度，单位 px |
