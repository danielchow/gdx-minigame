# CustomAd.onResize(function listener) ## # 功能描述
监听原生模板广告宽高回调事件（部分横幅模板支持）。
## 参数
### function listener
原生模板广告宽高回调事件的监听函数
#### 参数
##### Object res | 属性 | 类型 | 说明 |
| --- | --- | --- |
| width | number | 缩放后的宽度 |
| height | number | 缩放后的高度 |
