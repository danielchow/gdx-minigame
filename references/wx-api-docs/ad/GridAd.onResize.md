# GridAd.onResize(function listener)
从基础库 [2.30.2](../../guide/runtime/client-lib/compatibility.html) 开始，本接口停止维护
## 功能描述
监听 grid(格子) 广告尺寸变化事件。
## 参数
### function listener
grid(格子) 广告尺寸变化事件的监听函数
#### 参数
##### Object res | 属性 | 类型 | 说明 |
| --- | --- | --- |
| width | number | 缩放后的宽度 |
| height | number | 缩放后的高度 |
