# RewardedVideoAd.onClose(function listener) ## # 功能描述
监听用户点击 `关闭广告` 按钮的事件。
## 参数
### function listener
用户点击 `关闭广告` 按钮的事件的监听函数
#### 参数
##### Object res | 属性 | 类型 | 说明 | 最低版本 |
| --- | --- | --- | --- |
| isEnded | boolean | 视频是否是在用户完整观看的情况下被关闭的 | 2.1.0 |
