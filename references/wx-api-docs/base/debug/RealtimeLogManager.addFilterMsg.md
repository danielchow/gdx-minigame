# RealtimeLogManager.addFilterMsg(string msg)
基础库 2.14.4 开始支持，低版本需做[兼容处理](../../../guide/runtime/client-lib/compatibility.html)。

相关文档: [实时日志](../../../guide/runtime/debug/realtimelog/index.html)
## 功能描述
添加过滤关键字，暂不支持在插件使用
## 参数
### string msg
是setFilterMsg的添加接口。用于设置多个过滤关键字。
