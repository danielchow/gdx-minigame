# UpdateManager wx.getUpdateManager()
基础库 1.9.90 开始支持，低版本需做[兼容处理](../../../guide/runtime/client-lib/compatibility.html)。

**微信 Windows 版**：支持

**微信 鸿蒙 OS 版**：支持
## 功能描述
获取**全局唯一**的版本更新管理器，用于管理小程序更新。关于小程序的更新机制，可以查看[运行机制](../../../guide/runtime/operating-mechanism.html)文档。
## 返回值
### UpdateManager
更新管理器对象
