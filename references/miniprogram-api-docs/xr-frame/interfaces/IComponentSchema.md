## Interface: IComponentSchema

`Component`属性的注解接口。

`key`是可以写在组件对应于`xml`中的属性的名字。
`type`是属性的类型，由[registerDataValue](./../modules.html#registerDataValue)注册。
可选的`defaultValue`可以定义默认值。
