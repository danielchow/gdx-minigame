## # Properties

### # abortAudio

• `Optional` **abortAudio**: `boolean`

是否禁止音频，默认禁止。

### # autoPause

• `Optional` **autoPause**: `boolean`

是否在小程序压后台时自动暂停，默认暂停。

### # autoPlay

• `Optional` **autoPlay**: `boolean`

是否要在加载完毕后自动播放。

### # loop

• `Optional` **loop**: `boolean`

是否要循环播放。

### # placeHolder

• `Optional` **placeHolder**: [`IImage`](./IImage.html)

视频未加载成功时，可选的首帧图片地址。

### # src

• **src**: `string`

视频地址。
