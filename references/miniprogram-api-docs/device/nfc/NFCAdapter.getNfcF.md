## NfcF NFCAdapter.getNfcF()

基础库 2.11.2 开始支持，低版本需做[兼容处理](../../../framework/compatibility.html)。

**小程序插件**：支持

**微信 iOS 版**：不支持

**微信 Android 版**：支持

**微信 鸿蒙 OS 版**：支持

相关文档: [近场通信 (NFC)](../../../framework/device/nfc.html)

获取NfcF实例，实例支持NFC-F (JIS 6319-4)标准的读写
