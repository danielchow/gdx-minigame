## VKSession.update3DMode(Object object)

基础库 2.30.2 开始支持，低版本需做[兼容处理](../../../framework/compatibility.html)。

**小程序插件**：支持，需要小程序基础库版本不低于 [2.30.2](../../../framework/compatibility.html)

## # 功能描述

更新三维识别相关配置，要求调 [wx.createVKSession](wx.createVKSession.html) 时使用 face / hand / body。
