## VKSession.updateMaskMode(Object object)

基础库 3.2.1 开始支持，低版本需做[兼容处理](../../../framework/compatibility.html)。

**小程序插件**：支持，需要小程序基础库版本不低于 [3.2.1](../../../framework/compatibility.html)

设置裁剪相关配置，要求调 [wx.createVKSession](wx.createVKSession.html) 时使用 shoe。
