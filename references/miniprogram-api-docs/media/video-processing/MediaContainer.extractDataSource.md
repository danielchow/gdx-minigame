## MediaContainer.extractDataSource(Object object)

基础库 2.9.0 开始支持，低版本需做[兼容处理](../../../framework/compatibility.html)。

**小程序插件**：支持

## # 功能描述

将传入的视频源分离轨道。不会自动将轨道添加到待合成的容器里。
