## VideoContext.playbackRate(number rate)

基础库 1.4.0 开始支持，低版本需做[兼容处理](../../../framework/compatibility.html)。

**小程序插件**：支持

**微信 鸿蒙 OS 版**：支持

相关文档: [video 组件](../../../component/video.html)

## # 功能描述

设置倍速播放

## # 参数

### # number rate

倍率，支持 0.5/0.8/1.0/1.25/1.5，2.6.3 起支持 2.0 倍速，3.7.5 起支持 3.0/4.0 倍速
