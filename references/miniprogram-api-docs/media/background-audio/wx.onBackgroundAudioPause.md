## wx.onBackgroundAudioPause(function listener)

从基础库 [1.2.0](../../../framework/compatibility.html) 开始，本接口停止维护，请使用 [wx.getBackgroundAudioManager](wx.getBackgroundAudioManager.html) 代替

**小程序插件**：不支持

**微信 鸿蒙 OS 版**：支持

监听音乐暂停事件。
