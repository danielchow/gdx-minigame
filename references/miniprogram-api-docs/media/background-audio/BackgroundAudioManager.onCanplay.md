## BackgroundAudioManager.onCanplay(function listener)

**小程序插件**：支持

**微信 鸿蒙 OS 版**：支持

## # 功能描述

监听背景音频进入可播放状态事件。 但不保证后面可以流畅播放

## # 参数

### # function listener

背景音频进入可播放状态事件的监听函数
