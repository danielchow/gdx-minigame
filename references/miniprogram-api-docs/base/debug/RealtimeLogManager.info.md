## RealtimeLogManager.info()

基础库 2.7.1 开始支持，低版本需做[兼容处理](../../../framework/compatibility.html)。

**小程序插件**：不支持

相关文档: [实时日志](../../../framework/realtimelog/index.html)

## # 功能描述

写 info 日志，暂不支持在插件使用

## # 参数

### # Object|Array.<any>|number|string ...args

日志内容，可以有任意多个。每次调用的参数的总大小不超过5Kb
