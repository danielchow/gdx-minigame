## wx.cancelIdleCallback(number idleCallbackId)

基础库 3.10.0 开始支持，低版本需做[兼容处理](../../../framework/compatibility.html)。

**小程序插件**：不支持

**微信 Windows 版**：支持

**微信 Mac 版**：支持

**微信 鸿蒙 OS 版**：支持

## # 功能描述

取消之前注册的指定回调函数

## # 参数

### # number idleCallbackId

## # 示例代码

```js
wx.cancelIdleCallback(idleCallbackId)
```
