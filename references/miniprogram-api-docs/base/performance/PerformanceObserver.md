## PerformanceObserver

基础库 2.11.0 开始支持，低版本需做[兼容处理](../../../framework/compatibility.html)。

PerformanceObserver 对象，用于监听性能相关事件

## # 属性

### # Array supportedEntryTypes

获取当前支持的所有性能指标类型

## # 方法

### # PerformanceObserver.observe(Object options)

开始监听

### # PerformanceObserver.disconnect()

停止监听
