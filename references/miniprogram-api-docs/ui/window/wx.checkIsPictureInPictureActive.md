## boolean wx.checkIsPictureInPictureActive()

基础库 2.29.2 开始支持，低版本需做[兼容处理](../../../framework/compatibility.html)。

**小程序插件**：不支持

**微信 鸿蒙 OS 版**：支持

## # 功能描述

返回当前是否存在小窗播放（小窗在 video/live-player/live-pusher 下可用）

## # 返回值

### # boolean

当前是否存在小窗播放
