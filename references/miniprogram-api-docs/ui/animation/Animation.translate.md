**小程序插件**：支持

相关文档: [动画](../../../framework/view/animation.html)

## # 功能描述

平移变换

## # 参数

### # number tx

当仅有该参数时表示在 X 轴偏移 tx，单位 px

### # number ty

在 Y 轴平移的距离，单位为 px
