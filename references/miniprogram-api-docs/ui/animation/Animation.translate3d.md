## Animation Animation.translate3d(number tx, number ty, number tz)

**小程序插件**：支持

相关文档: [动画](../../../framework/view/animation.html)

## # 功能描述

对 xyz 坐标进行平移变换

## # 参数

### # number tx

在 X 轴平移的距离，单位为 px

### # number ty

在 Y 轴平移的距离，单位为 px

### # number tz

在 Z 轴平移的距离，单位为 px
