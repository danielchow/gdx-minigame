## Animation Animation.skew(number ax, number ay)

**小程序插件**：支持

相关文档: [动画](../../../framework/view/animation.html)

## # 功能描述

对 X、Y 轴坐标进行倾斜

## # 参数

### # number ax

对 X 轴坐标倾斜的角度，范围 [-180, 180]

### # number ay

对 Y 轴坐标倾斜的角度，范围 [-180, 180]
