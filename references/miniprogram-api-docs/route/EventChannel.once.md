## EventChannel.once(string eventName, function fn)

基础库 2.7.3 开始支持，低版本需做[兼容处理](../../framework/compatibility.html)。

**小程序插件**：支持

## # 功能描述

监听一个事件一次，触发后失效

## # 参数

### # string eventName

事件名称

### # function fn

事件监听函数

#### # 参数

##### # any args

触发事件参数
