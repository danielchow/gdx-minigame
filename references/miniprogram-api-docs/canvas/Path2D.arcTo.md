## Path2D.arcTo(number x1, number y1, number x2, number y2, number radius)

基础库 2.11.0 开始支持，低版本需做[兼容处理](../../framework/compatibility.html)。

**小程序插件**：不支持

相关文档: [画布指南](../../framework/ability/canvas.html)、[canvas 组件介绍](../../component/canvas.html)

## # 功能描述

通过给定控制点添加一段圆弧路径。

## # 参数

### # number x1

第一个控制点横坐标。

### # number y1

第一个控制点纵坐标。

### # number x2

第二个控制点横坐标。

### # number y2

第二个控制点纵坐标。

### # number radius

圆形半径，必须为非负数。
