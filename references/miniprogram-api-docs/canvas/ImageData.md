ImageData 对象

## # 属性

### # number width

使用像素描述 ImageData 的实际宽度

### # number height

使用像素描述 ImageData 的实际高度

### # Uint8ClampedArray data

一维数组，包含以 RGBA 顺序的数据，数据使用 0 至 255（包含）的整数表示
